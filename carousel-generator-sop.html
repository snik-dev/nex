Standard-Operating Procedure
MNE.com Carousel Builder

1. Getting started

  1. Open carousel-builder.html in Chrome.
  2. (Optional) paste a Carousel Title; it renders as the blue headline on site.
  3. Review the global field toggles: Retail, NEX Price, Sale Price, Reviews, Badge, Brand.

2. Adding individual products

  1. Click Add Product. A card opens with editable inputs.
  2. Fill Product URL (relative OK), Image URL, and any optional fields.
    • When Sale Price is present, Retail and NEX values will auto-strike and the sale displays red.
  3. Collapse a card by clicking its title row; only one card stays open to keep the page tidy.

3. Bulk CSV upload

  1. Click Bulk CSV Upload in the blue header.
  2. Select a file that matches the template column order.
  3. Rows import instantly; edit any cell before export.

4. Importing existing HTML

  1. Paste legacy snippet into the Paste HTML to Import box.
  2. Click Import. Builder extracts brand, prices, rating GIF, reviews, badge, and images.

5. Live preview & re-ordering

  • Preview updates in real-time with ★/☆ rating symbols.
  • Drag a preview tile to change order; form rows sync automatically.

6. Generating the embed code

  1. Scroll to Generated HTML.
  2. Click anywhere, press Ctrl / Cmd +A then Ctrl / Cmd +C – or highlight and copy.
  3. Paste directly into the CMS module. The block already contains the Owl Carousel init script.

7. Quick tips

  • Max 10 items per carousel.
  • Image URLs must point to the /products/images/small/ path for proper CDN sizing.
  • Promo badge text follows: Clearance · Last Chance · New · NEX Exclusive.
  • You can reorder via keyboard: Ctrl / Cmd + ↑ / ↓ when a card header is focused.
